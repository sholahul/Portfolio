Sentiment Analysis of Informatics Using Naive Bayes Method


Informatics is one of the majors that exist in the Computer Science Faculty of Sriwijaya University. The major was established in 2004 and has more than 400 students. The number of students who have or are studying in the informatics make a lot of opinions formed about how it feels to study in this major. Our research will group these opinions. This study utilizes the Naive Bayes method for grouping those opinions.

  • Functional Requirements

    The user is able to input the opinion and
    The system is able to group the opinion into positive, negative, and neutral

  • Non-Functional Requirements

    Performance
    The system is able to analyze up to 500 words and
    Receive document format only in .txt form
    Accessibility
    The website implements the display with Bootstrap framework

  • Use Case

    The system is able to group the opinions and
    The system is able to do the pre-processing
